🤖 과거 전국 기능경기대회 모바일 로보틱스 은메달 수상 경험이 있습니다.대회 경험을 통해 로직 설계와 문제 해결 과정에 흥미를 가지게 되었고  현재는 이를 백엔드 개발 역량으로 확장하고 있습니다.
